# Мини-чат
## Установка и запуск
- устанавливаем необходимые пакеты: ```npm install```
- запускаем сервер: ```node server-socket.js```

После приложение будет доступно по адресу __http://localhost:8080__

## Описание

Приложение позволяет обмениваться сообщениями в реальносм времени на основе протокола веб-сокетов.
